## HALS – Гиперспектральная автоматизированная летная система

![Логотип проекта](GUI/Icons/hals_image.png)

**HALS** — программное обеспечение для управления гиперспектральной камерой, обзорной камерой, GPS-приёмником и модулем стабилизации в составе беспилотной авиационной системы. Проект разрабатывается в НИИПФП им. А.Н. Севченко БГУ и предназначен для регистрации гиперспектральных данных, их интеллектуальной обработки в реальном времени и геопространственной привязки.

---

## 🚀 Возможности (в разработке)

- **Управление двумя камерами Basler** (гиперспектрометр и обзорная) в режиме master-slave с аппаратной синхронизацией.
- **Асинхронный захват и обработка** изображений с использованием многопоточности (`QThreadPool`, `QRunnable`, `QtConcurrent`).
- **Динамическое изменение параметров** камер с автоматической корректировкой взаимосвязанных настроек (ROI, биннинг, смещение, экспозиция, частота кадров).
- **Тепловая карта гиперспектральных данных** – визуализация спектров с разбиением на поддиапазоны (настраиваемое количество) и выбором цветовой палитры (Viridis, Cividis, Plasma, Grayscale и др.).
- **Отображение поля зрения спектрометра** на изображении обзорной камеры в виде полупрозрачного прямоугольника (координаты настраиваемые).
- **Приём и парсинг данных GPS** (формат NMEA, модуль U‑blox 7) с выдачей геопривязанной временной метки.
- **Асинхронное сохранение** данных в форматах BMP и бинарных файлов (raw) с выбором сохраняемых камер.
- **Графический интерфейс** с тёмной темой, отображением видеопотоков (обзорная камера и тепловая карта), настройкой параметров и индикацией состояния устройств (USB, GPS, температура CPU).
- **Модульная архитектура** с полной поддержкой кроссплатформенности – успешно портировано на Raspberry Pi 5 (Raspbian / Linux).
- **Асинхронное логирование** с ротацией файлов и контролем температуры процессора.

## 🧪 Сборка и запуск

### Требования
- **Qt 6.5+** (с модулями: core, gui, widgets, concurrent, serialport)
- **Pylon SDK 6.3.0** (для Windows) или соответствующая версия для Linux (arm64)
- Компилятор с поддержкой C++17

### Инструкция (Qt Creator)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Lom-Karatist/HALS.git
   ```
2. Откройте файл `HALS.pro` в Qt Creator.
3. Убедитесь, что пути к Pylon SDK в `Basler/Basler.pri` настроены под вашу систему.
4. Выберите конфигурацию сборки (Debug/Release) и нажмите **Build**.
5. Запустите приложение.

> **Примечание:** Для работы с камерами требуется физическое подключение двух Basler acA1920-40um и кабель синхронизации (пин 3 → пин 4).
